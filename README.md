<h2>Working Of Bot</h2>
<p>This is A DIscord Bot Which Plays Music With The help of a Python Library Named "YT_DLP" it is a free library available for Every One to Use and the Songs are PLayed From FFMPEG(A Software Free On Inrernet) in The System and then The Music Streams On Discord Voice Channel With the Help of PyNacl(Python Library). </p>
<h2>Config.yml</h2>
<p>
  Config.yml has some options like
  <li>Bot Name: "You Can Write Bots Name Here"</li>
  <li>Bot prefix: "?"</li>
  <li>Embed Color: "hex code" eg: 0x0FFFF</li>
  <li>Token: "Your Bot Token You Got From the Discord Dev Portal"</li>
</p>

# Commands
1. `play <song name>` <br>
2. `skip` skips a song <br>
3. `stop` stops the current playing song

Currently Only These Commands are Supported
